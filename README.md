# 🐉 Demogorgon Tamagotchi (Pet-Demo)

A cute but dangerous virtual pet inspired by *Stranger Things*. Take care of your little monster directly in your browser!

## 🎮 Features
* **Interactivity**: You can feed your Demogorgon and play with it.
* **Mood System**: The UI uses emojis to show how he feels.
* **Consequences**: 
    * If you don't feed him, he gets **angry** 😡.
    * If you neglect him for too long, he will **escape back to Hawkins** 🌲.

## 🛠️ Tech Stack
* **JavaScript** (Logic & Mechanics)
* **HTML/CSS** (Interface)

## 📝 Note
The interface is currently in **Russian**, but the gameplay is intuitive thanks to the emoji system.
